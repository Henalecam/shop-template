# Admin Frontend

- NEXT_PUBLIC_API_URL: URL da API do backend (ex.: `https://seu-backend.up.railway.app/api`).
- NEXT_PUBLIC_FRONTEND_BASE_URL (opcional): URL pública do template frontend para abrir preview com `?tenantId=`.

## Desenvolvimento
```bash
npm install
npm run dev
```

## Produção
```bash
npm run build
npm start
```
